# 小说 AI 工作台 · 使用说明

本指南介绍应用的主要功能与操作方式。安装相关请看 [安装说明](安装说明.md)。

> 所有数据都保存在你本机的一个数据库文件中(可在「设置 → 数据存放位置」更改位置),不会上传到任何服务器。

---

## 一、作品管理(主页)

打开应用后进入主页,这里管理你的所有小说作品。

- **创建新书**:点击作品网格中的「**+**」卡片新建一本小说。
- **打开作品**:点击作品卡片,进入该书的写作工作区。
- **作品卡片**:显示书名、最近更新时间,右上角有删除按钮。
- **删除作品**:点卡片上的垃圾桶图标,作品进入**回收站**(不会立即丢失)。
- **回收站**:点顶部的「回收站」链接进入,可对已删除作品执行:
  - **恢复**:还原回作品列表;
  - **永久删除**:彻底删除,**不可恢复**。

---

## 二、章节与写作

进入一本书后,即可管理章节并开始写作。

### 章节管理
- **新建章节**:左侧栏或头部的「**+ 新建章节**」按钮。
- **章节列表**:显示每章的序号、标题、字数、更新时间和状态(草稿 / 审阅 / 定稿)。
- **章节 / 回收站标签**:在头部切换查看正常章节或已删除章节。
- **删除 / 恢复 / 永久删除章节**:章节卡片上的「X」删除进回收站;回收站里可恢复或彻底删除。
- **统计**:左侧栏顶部显示「X 章 · Y 字」的总览。

### 正文编辑器
- **编辑 / 阅读 两种模式**:阅读模式点头部按钮切到编辑模式修改标题和正文。
- **章节标题**:写作区顶部输入本章标题。
- **自动分段与字数统计**:回车自动分段,头部实时显示「正文 X 字」。
- **保存状态**:头部显示自动保存间隔 / 已保存状态(自动保存间隔可在菜单「设置 → 自动保存」调整)。

## 三、AI 辅助写作

> AI 功能需要先在「**设置 → AI 连接**」里配置好你的 AI 接口(见下文第八节)。未配置也能正常写作,只是 AI 功能不可用。

在写作界面可对**选中文本**或**整章**调用 AI:

| 功能 | 作用 |
|------|------|
| **续写** | 根据锚点前文继续往下写,可插入到光标处或追加到正文末尾 |
| **改写** | 改写选中文本或整章,可指定风格、节奏或视角 |
| **润色** | 优化文字细节,增强画面感与质感 |
| **全章生成** | 参考前 5 章的时间线、伏笔、人物关系和资料库,生成整章内容 |
| **灵感** | 生成后续写作方向与灵感建议,帮你突破卡文(只给方向,不直接改正文) |
| **AI 纠错** | 识别并修正错别字、漏字、多字、误用字和标点错误 |
| **AI 一键总结** | 串行同步时间线、创作计划、人物关系图和资料库人物信息 |
| **引用参考资料** | 写作时引入资料库、伏笔、创作计划等内容供 AI 读取参考 |

## 四、人物与设定资料库(百科)

为作品建立「人物」与「设定(世界观)」资料,供查阅和 AI 参考。

- **新建条目**:头部「**+ 新建**」按当前分区创建人物或设定。
- **人物 / 设定 切换**:头部切换两个分区,并显示各自数量。
- **搜索**:按名称、类型或摘要文字筛选。
- **编辑 / 删除**:选中条目后在详情头部操作。
- **人物字段**:角色名称(必填)、角色类型(主角/配角/反派/导师/盟友/恋人/家人/对手/神秘人/其他)、简介、详细信息(背景、性格、外貌)。
- **设定字段**:标题(必填)、类型(地点/组织/物品/事件/种族/文化/魔法·能力/科技/其他)、详细内容、标签(逗号分隔)。
- 写作界面左侧的**资料库**按人物 / 背景 / 设定 / 其他分组管理;底部「**管理百科 →**」可进入集中管理页。

## 五、写作工作区的功能标签

进入一本书后,工作区顶部有多个标签页:

### 规划
- **故事总览**:查看和编辑整本书的大纲;
- **章节摘要**:当前章节摘要 + 全书摘要索引;
- **创作计划**:为每章制定计划并跟踪完成状态,「新建计划」添加计划项。

### 伏笔库
管理「埋坑 → 回收」的完整生命周期。
- **新建伏笔**、按状态**筛选**(全部 / 未回收 / 推进中 / 已回收);
- **伏笔详情**:内容、备注、首次埋设章节、回收章节、状态;
- **章节响应记录**:伏笔在各章的推进、线索、响应与回收。

### 人物关系
- **资料库关系图**:所有人物的长期关系网络;
- **当前章节关系图**:本章涉及人物的关系,可点 **「AI 更新章节关系图」** 由 AI 根据正文自动分析生成。

### 时间线
可视化整本书的时间脉络,逐章记录:
- **具体时间**(含穿越前后 / 回忆 / 倒叙 / 相对时间)、**剧情事件**、**人物状态**(情绪/目标/立场/身体状态)、**章节总结**(供 AI 长期记忆)、**伏笔回收**;
- 点 **「AI 更新章节时间线」** 由 AI 根据正文自动生成以上各项。

### 对话
与 AI 进行通用对话,讨论剧情、人物、设定或复盘章节。

## 六、菜单与设置

顶部为深色菜单栏(文件 / 编辑 / 视图 / 窗口 / 帮助 / 设置)。

- **设置 → 自动保存**:关闭 / 5 / 10(默认)/ 30 / 60 秒。
- **设置 → 数据存放位置**:更改数据库存放文件夹、在文件管理器中打开、恢复默认位置(详见 [安装说明](安装说明.md) 的「数据保存在哪里」)。
- **编辑**:撤销 / 重做 / 剪切 / 复制 / 粘贴 / 全选(也支持标准快捷键 Ctrl+Z/Y/X/C/V/A)。
- **视图**:重新加载、开发者工具、缩放、全屏(F11)。

## 七、配置 AI(让 AI 功能可用)

AI 功能使用**你自己的** AI 接口,应用本身不含 AI 额度。

1. 进入任意一本书的写作界面,打开 **「设置」** 标签 → **「AI 连接」**。
2. 选择连接方式:
   - **添加 OpenAI**:官方 OpenAI API;
   - **添加 OpenAI 兼容连接**:任何支持 `/chat/completions` 的服务(如 DeepSeek、国内中转、本地大模型等)。
3. 填写:
   - **连接名称**:便于识别的名字;
   - **Base URL**:API 服务器地址;
   - **API Key**:认证密钥(仅保存在本机);
   - **支持的模型**:拉取并选择默认模型。
4. 在 **「提示词管理」** 中,可统一自定义对话、续写、改写、润色、全章、灵感、纠错、一键总结、关系图、时间线等各类 AI 的提示词。

## 八、数据安全小贴士

- 所有作品都在本机数据库文件 `novel-ai-studio.db` 中;**关闭应用后直接复制该文件即是一份完整备份**。
- 数据位置、备份与迁移的详细说明见 [安装说明](安装说明.md)。
- 若把数据放到云同步盘,**请勿在多台电脑上同时编辑**,以免数据库冲突损坏。
