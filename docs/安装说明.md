# 小说 AI 工作台 · 安装说明

一款本地运行的 Windows 桌面写作工具:章节管理、人物/设定百科、伏笔与时间线、AI 辅助写作。所有作品数据都保存在你自己的电脑上。

---

## 一、系统要求

- **操作系统**：Windows 10 / Windows 11(64 位）
- **磁盘空间**：约 300 MB
- **联网**：本体离线可用；只有使用「AI 辅助写作」时需要联网（调用你自己配置的 AI 服务）

---

## 二、下载

1. 打开发布页面：
   **https://github.com/ideal-ai-mu/novel-ai-platform/releases**
2. 在最新版本（如 `v1.0.0`）下，下载安装包：
   **`Novel-AI-Studio-Setup-1.0.0.exe`**

---

## 三、安装

1. 双击下载好的 `Novel-AI-Studio-Setup-1.0.0.exe`。

2. **如果弹出蓝色提示「Windows 已保护你的电脑 / Windows protected your PC」**：
   这是因为本安装包暂未购买代码签名证书，并非病毒。处理方法：
   - 点击提示里的 **「更多信息 / More info」**
   - 再点 **「仍要运行 / Run anyway」**

   > 杀毒软件（如 360、火绒、Defender）偶尔也会因「未签名」误报，遇到时选择「允许 / 信任」即可。

3. 按向导完成安装：
   - 可以选择安装位置（默认即可）
   - 安装完成后会在桌面和开始菜单生成「**Novel AI Studio**」快捷方式

4. 启动应用——首屏标题是「**小说 AI 工作台**」即表示安装成功。

---

## 四、首次使用

### 1. 创建你的第一本书
- 首页点击 **「创建新书」**，填写书名即可进入。

### 2.（可选）配置 AI 服务
AI 功能需要你自己的 AI 接口（应用本身不含 AI 额度）。

- 进入任意一本书的**写作界面**，打开 **「AI 连接设置」** 面板；
- 选择一种连接方式：
  - **官方 OpenAI 连接**：使用 OpenAI 官方接口；
  - **OpenAI 兼容连接**：任何兼容 `/chat/completions` 的服务（国内中转、本地大模型等）；
- 填写 **API 地址（Base URL）**、**API Key**，然后**选择模型**；
- 保存后即可在写作时使用续写、润色、灵感提示等 AI 功能。

> 不配置 AI 也能正常用来写作、管理章节与设定，只是 AI 辅助功能不可用。

### 3. 自动保存
- 顶部菜单 **设置 → 自动保存**，可选择关闭 / 5 / 10 / 30 / 60 秒（默认 10 秒）。

## 五、数据保存在哪里 · 如何备份/迁移

### 默认位置
所有作品（章节、人物、设定、伏笔、时间线）都保存在**同一个数据库文件**里：

```
C:\Users\你的用户名\AppData\Roaming\novel-ai-platform\novel-ai-studio.db
```

> 在地址栏输入 `%APPDATA%\novel-ai-platform` 可直接打开该文件夹。

### 更改存放位置（如放到 D 盘 / 移动硬盘 / 同步盘）
顶部菜单 **设置 → 数据存放位置**：
- **更改位置…**：选择一个**空文件夹**，程序会安全地把数据搬过去（复制并校验通过后才删除原文件）；
- **在文件管理器中打开**：快速打开当前数据所在文件夹；
- **恢复默认位置**：把数据移回默认目录。

> ⚠️ 放到 OneDrive / 坚果云 / Dropbox 等**云同步文件夹**时，请不要在两台电脑上同时打开并编辑同一份数据，否则可能导致数据库损坏。

### 备份
关闭应用后，直接复制 `novel-ai-studio.db` 这个文件保存到别处即可，就是一份完整备份。恢复时把它放回数据目录覆盖即可。

## 六、更新到新版本

1. 到 Releases 页面下载更新的安装包；
2. 直接安装（覆盖旧版本即可，**作品数据不会丢失**，因为数据存放在独立的数据目录中）。

---

## 七、卸载

- 在 **Windows 设置 → 应用** 或 **控制面板 → 程序和功能** 中卸载「**Novel AI Studio**」。
- 卸载**不会删除你的作品数据**。如需彻底清除，再手动删除数据目录
  `%APPDATA%\novel-ai-platform`（删除前请确认已备份）。

## 八、常见问题

**Q：安装/打开时被 SmartScreen 或杀毒拦截？**
A：因为暂未做代码签名，属正常现象。按「更多信息 → 仍要运行」，或在杀毒软件中信任即可。

**Q：打开后是白屏 / 报「初始化失败」？**
A：常见原因是数据存放位置不可用（比如之前把数据放到了已拔出的 U 盘/移动硬盘）。处理：重新连接该磁盘后重启应用；或点顶部菜单 **设置 → 数据存放位置 → 恢复默认位置**。你的数据仍在原来的文件夹里，不会丢。

**Q：AI 功能没反应 / 报错？**
A：检查「AI 连接设置」里的 API 地址、API Key 是否正确，模型是否已选择，以及网络是否能访问该接口。

**Q：找不到我的作品？**
A：确认数据存放位置（设置 → 数据存放位置 → 在文件管理器中打开），确认 `novel-ai-studio.db` 是否在那里。
